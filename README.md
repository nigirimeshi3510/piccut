# PicCut

ブラウザで使える 16:5 固定アスペクト比の画像切り抜きツールです。ブラウザが画像として開けるファイルを読み込み、表示用キャンバス上の切り抜き枠をドラッグして位置調整し、その範囲を元画像座標で正しくクロップして PNG 保存します。

## 機能

- ブラウザが認識できる画像ファイルを読み込み
- キャンバスに画像を表示
- 16:5 固定比率の切り抜き枠を表示
- マウスやタッチで枠をドラッグ移動
- 枠が画像外に出ないよう制約
- 元画像座標でクロップして PNG 保存
- オプションで `640x200` にリサイズして保存
- GUI上で出力ファイル名を編集可能
